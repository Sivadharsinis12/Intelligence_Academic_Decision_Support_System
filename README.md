# 🎓 Intelligent Academic Decision Support System (IADSS)

## 📌 Overview
The Intelligent Academic Decision Support System (IADSS) is a web-based application that assists academic institutions in making data-driven decisions based on student performance, attendance, and subject analytics.

The system provides dashboards, student risk detection, subject-level insights, and historical report tracking.

---

## 🚀 Features

- 📊 Academic Dashboard
- 📈 Student Performance Analytics
- ⚠ Risk Detection (High / Medium / Low)
- 🗂 Search & Filter History Tracking
- 🧠 Rule-based Risk Calculation Engine
- 🔄 Dynamic Dropdown Filtering
- 📡 RESTful API using FastAPI
